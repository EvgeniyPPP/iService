# 🔧 I Service — Сервісний центр

Веб-застосунок сервісного центру з ремонту техніки Apple.  
Дипломний проект за спеціальністю 123 Комп'ютерна інженерія.

---

## 🛠Технологічний стек

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express.js
- **База даних:** SQLite (better-sqlite3)
- **Авторизація:** JWT + bcryptjs

---

## Функціонал

- Головна сторінка з описом сервісу та послугами
- Сторінка "Про нас" з історією та командою
- Каталог послуг та прайс-лист (27 моделей iPhone × 10 послуг)
- Калькулятор вартості ремонту з підтримкою кількох послуг
- Реєстрація та авторизація клієнтів
- Особистий кабінет клієнта
- Створення заявок на ремонт
- Відстеження статусу замовлення по етапах
- Панель менеджера (керування замовленнями, клієнтами, послугами)
- Сторінка контактів

## Як запустити проект

### 1. Встановіть Node.js
Завантажте з [nodejs.org](https://nodejs.org) версію LTS

### 2. Клонуйте репозиторій
```bash
git clone https://github.com/EvgeniyPPP/iservice.git
cd iservice
```

### 3. Встановіть залежності
```bash
npm install
```
